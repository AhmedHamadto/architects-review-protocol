# Changelog

All notable changes to the Architect's Review Protocol are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-10

### Added
- Initial release of the 3-phase architectural review protocol
- Phase 1: Mapper protocol — structural mapping via import graph and interface extraction
- Phase 2: Reviewer protocol — tiered chunk-level review (Tier 1/2/3) with boundary verification
- Phase 3: Synthesizer protocol — cross-boundary analysis, deduplication, risk scoring, and final report
- Adaptation rules for small (<30 files), medium (30-150), large (150-500), and mega (500+) projects
- Resumption support for interrupted reviews
- Anti-patterns guide
